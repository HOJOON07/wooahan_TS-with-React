### 7. ESLint

![Untitled](https://velog.velcdn.com/images/hyohyo__/post/0afe9c92-8a57-40f3-b50c-5ade6ce190ee/image.png)

- 코드를 형식화하고 오류가 날 경우 경고 메세지를 지원함
- 코드의 일관성을 높이고 협업에 유용함
- 사용방법

1. ESLint 설치
2. npm install eslint 실행
3. npm init @eslint/config@latest 실행
4. 폴더에 eslint.config.js (또는 eslint.config.mjs) 생성되었는지 확인
5. .eslint.config.js의 rules에 원하는 규칙을 입력
